# 製品マニュアル v1.0

- 価格：月額 15,000円（税抜）
- 対応ユーザー数：最大50名
- サポート：平日9:00〜18:00（メール・電話）
- 主な機能：データ分析、レポート自動生成、API連携

## 製品B（プレミアムプラン）
- 価格：月額 30,000円（税抜）
- 対応ユーザー数：無制限
- サポート：24時間365日（専任担当者付き）
- 主な機能：製品Aの全機能 + AIによる予測分析、カスタムダッシュボード

## よくある質問

Q: 無料トライアルはありますか？
A: はい、14日間の無料トライアルをご用意しています。クレジットカード不要です。

Q: 既存システムとの連携は可能ですか？
A: REST APIを提供しており、主要なCRM・ERPシステムとの連携が可能です。

Q: データのバックアップはどうなりますか？
A: 毎日自動バックアップを実施。過去30日分のデータを復元可能です。
